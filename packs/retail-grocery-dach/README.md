# Discount Grocery Retail pack (DACH)

An industry pack for OpenConstructionERP that pre-configures the platform for
German-speaking discount food-retail new-builds.

It ships three fully priced example projects, each a DIN 276 cost plan with a
complete Leistungsverzeichnis across the retail trades:

- Lebensmittelmarkt Heilbronn (flagship, urban edge site)
- Lebensmittelmarkt Heidelberg (constrained urban infill)
- Lebensmittelmarkt Karlsruhe (larger greenfield format)

Each project carries earthworks, shell and foundations, roof, precast frame,
facade, windows and doors, drywall and interior fit-out, mechanical and
sanitary, CO2 refrigeration, electrical, photovoltaics with battery storage,
external works and parking, signage and store fit-out, priced in EUR with a
site-overhead markup and 19 percent VAT.

The pack defaults to EUR, German locale and the DACH standards stack: DIN 276,
GAEB DA XML 3.3, an LV-quality rule set and BKI plausibility benchmarks.
